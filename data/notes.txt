Again, when it comes to the order of the values, I see no reason not to use the order given by the W3C. Better safe than sorry.

When using the font shorthand you can omit any values except font-size and font-family – 
you always need to give values for those, and in that order. 

The initial values for the individual font properties are these:

    * font-style: normal
    * font-variant: normal
    * font-weight: normal
    * font-size: medium
    * line-height: normal
    * font-family: depends on the user agent
